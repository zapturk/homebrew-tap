# zapturk Homebrew Tap

This is the official Homebrew tap for [zapturk](https://github.com/zapturk) tools.

## Installation

To add this tap to Homebrew, run:

```bash
brew tap zapturk/homebrew-tap
```

## Available Casks

### Go-RomM-Sync

[Go-RomM-Sync](https://github.com/zapturk/Go-RomM-Sync) is a desktop application to sync your RomM library with local devices.

To install:

```bash
brew install --cask go-romm-sync
```
